# Система контроля и управления доступом (СКУД)

Система контроля и управления доступом с контекстно-зависимой динамической аутентификацией на основе HMAC-SHA256 и ранговой системы.

## 📋 Описание

Прототип системы СКУД с оригинальным методом аутентификации на основе:
- **Ранговой комбинации** (покерные комбинации от 3 до 9)
- **Истории перемещений** (контекстная проверка маршрута)
- **HMAC-SHA256 токена** с криптографически стойким nonce
- **Защиты от replay-атак** и brute-force

## 🔑 Ключевые особенности

- ✅ Аутентификация на основе HMAC-SHA256
- ✅ Ранговая система (3-9) на основе покерных комбинаций
- ✅ Контекстно-зависимый контроль доступа
- ✅ История перемещений пользователей
- ✅ Защита от replay-атак (nonce)
- ✅ Блокировка после 3 неудачных попыток (1 минута)
- ✅ Веб-интерфейс для управления и мониторинга
- ✅ CSRF-защита форм
- ✅ Логирование всех событий
- ✅ Управление пользователями
- ✅ Интерактивная схема офиса (real-time)
- ✅ REST API для интеграции
- ✅ **NFC-интеграция с PN532** (Raspberry Pi)

## 🚀 Быстрый старт

### 1. Установка (Raspberry Pi + NFC)

```bash
cd /path/to/skud-diploma
sudo ./install.sh
sudo reboot
```

### 2. Подключение PN532

```
PN532          →    Raspberry Pi
─────────────────────────────────────
VCC            →    3.3V (Pin 1)
GND            →    GND (Pin 6)
SCL            →    GPIO 3 (Pin 5)     # I2C
SDA            →    GPIO 2 (Pin 3)     # I2C
```

### 3. Запуск

```bash
# Веб + NFC
./start.sh --nfc

# Только веб
./start_web.sh
```

**Откройте:** http://localhost:5000

## 🔧 Использование

### Регистрация пользователя

```bash
python3 personalize.py --uid ADMIN_01 --rank 9 --name "Администратор"
```

### Веб-интерфейс

- **Эмуляция прохода** — выбор пользователя или ручной UID
- **NFC считыватель** — кнопка "Приложить карту"
- **Пользователи** — управление (добавление/редактирование/удаление)
- **Схема офиса** — real-time отображение по зонам
- **Журнал** — история событий

### NFC Сервис

```bash
# Автономная обработка карт
sudo ./start_nfc.sh --zone 1

# С авто-регистрацией
sudo ./start_nfc.sh --zone 1 --auto-register --rank 4
```
