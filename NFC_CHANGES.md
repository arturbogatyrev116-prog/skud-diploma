## 🔐 Безопасность

### Изменения

- **Нет изменений в ядре безопасности** — NFC использует существующую HMAC-SHA256 аутентификацию
- **Изоляция процессов** — NFC сервис работает отдельно от веб-сервера
- **Минимальные привилегии** — systemd сервисы с ограничениями

### Рекомендации

1. Используйте **NTAG 424 DNA** для защищённых карт
2. Включите **HTTPS** для веб-интерфейса
3. Ограничьте **физический доступ** к Raspberry Pi
4. Настройте **брандмауэр** (UFW)

## 🧪 Тестирование

### Новые тесты

```bash
# Тест NFC модуля
python3 test_nfc.py

# Тест чтения UID
sudo python3 nfc_service.py --zone 1

# Тест с авто-регистрацией
sudo python3 nfc_service.py --zone 1 --auto-register --rank 4
```

### Проверка работы

1. **Базовое чтение** — UID карты определяется
2. **Аутентификация** — проверка прав доступа
3. **Запись NTAG** — данные записываются и читаются
4. **Непрерывный режим** — стабильный опрос без сбоев

## 🐛 Известные проблемы

1. **Эмуляция без железа** — работает только базовое чтение
2. **Длинные линии связи** — могут быть проблемы со SPI на расстоянии >20 см
3. **Металлические поверхности** — ухудшают чтение NFC

## 🔧 Настройки по умолчанию

### NFC Сервис

```python
zone_to = 1              # Зона "Офис"
auto_register = False    # Авто-регистрация выключена
default_rank = 4         # Ранг посетителя
read_cooldown = 2.0      # Задержка между чтениями (сек)
```

### SPI Настройки

```python
spi_bus = 0              # SPI шина 0
spi_device = 0           # SPI устройство 0 (CE0)
spi_speed = 500000       # 500 kHz
reset_pin = 17           # GPIO 17 (BCM)
irq_pin = 25             # GPIO 25 (BCM)
```
